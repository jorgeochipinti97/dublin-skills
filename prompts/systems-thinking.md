# systems-thinking

## Activation Prompts

```
Analyze [product/system] as a system with feedback loops
```

```
Find leverage points in [process]
```

```
Map the stocks and flows for [business]
```

```
Identify bottlenecks in [workflow]
```

```
Create a causal loop diagram for [problem]
```

## Example Use Cases

- Analyzing user growth dynamics (reinforcing loops)
- Finding performance bottlenecks (balancing loops)
- Mapping system behavior patterns
- Identifying high-leverage intervention points
- Understanding delays and their effects
