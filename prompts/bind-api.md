# bind-api

## Activation Prompts

```
Integrate with BIND Argentina API for [use case]
```

```
Create a transfer to CBU/CVU using BIND API
```

```
Set up DEBIN collection with BIND
```

```
Validate a CBU/CVU with BIND API
```

```
Issue an eCheq using BIND Open Banking
```

## Example Use Cases

- Querying account balances and transactions
- Making transfers via CBU or CVU
- Creating DEBIN (immediate debit) requests
- Issuing and depositing electronic checks (eCheqs)
- Validating CBU/CVU ownership
- Creating and managing CVUs for collections
