# brand-identity

## Activation Prompts

```
Define the brand identity for [product name]
```

```
Set up the color palette and typography for this project
```

```
What colors and fonts should I use for a [tech/luxury/friendly] brand?
```

```
Create brand guidelines for [company/project]
```

```
Review this design against our brand system
```

## Example Use Cases

- Starting a new project and need to establish visual identity
- Ensuring consistency across multiple components or pages
- Choosing color palettes and typography for a specific brand personality
- Validating that existing designs follow the established system
- Integrating with premium-frontend-design to constrain style choices
