# skill-creator

## Activation Prompts

```
I want to create a new skill for [domain/task]
```

```
Help me build a skill that handles [specific workflow]
```

```
Create a skill for working with [technology/API/format]
```

```
I need to package my existing skill at [path]
```

```
Create the prompt cheat sheet for [skill-name]
```

## Example Use Cases

- Creating a skill for PDF manipulation
- Building a skill for a specific API integration
- Packaging a workflow into a reusable skill
- Updating an existing skill with new capabilities
- Generating activation prompts for a new skill
