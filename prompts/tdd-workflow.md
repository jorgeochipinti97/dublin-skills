# tdd-workflow

## Activation Prompts

```
Implement [feature] using TDD
```

```
Write tests first for [functionality]
```

```
Guide me through red-green-refactor for [task]
```

```
Create test cases from these acceptance criteria
```

```
Help me write unit tests for [component]
```

## Example Use Cases

- Implementing a new feature with failing tests first
- Converting user stories to test cases (Given/When/Then)
- Writing unit tests with mocks and stubs
- Refactoring code while keeping tests green
- Setting up test doubles for dependencies
