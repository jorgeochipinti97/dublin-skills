# product-planner

## Activation Prompts

```
Create a PRD for [product idea]
```

```
Write user stories for [feature]
```

```
Define the MVP scope for [project]
```

```
Help me plan the product requirements for [concept]
```

```
Create acceptance criteria for [user story]
```

## Example Use Cases

- Writing a Product Requirements Document
- Breaking features into user stories with Given/When/Then
- Defining MVP vs v2 scope
- Setting success metrics (North Star, input metrics)
- Prioritizing backlog (Must/Should/Could/Won't)
